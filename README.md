# Derrubando da Torre de Marfim

> _"Um espaço para reflexão, ciência, sociedade e o livre debate de ideias."_

Este é o repositório do blog pessoal de **Wesley Gueta**. Este espaço nasce não sob a pretensão de deter certezas ou conhecimentos absolutos, mas sim como um diário de exploração e aprendizado de alguém profundamente curioso pelo mundo. O objetivo é compartilhar leituras, vivências e reflexões sobre ciência, filosofia e sociedade, construindo pontes para um diálogo horizontal no qual possamos debater e aprender juntos. Futuramente, pretendo implementar um sistema de federação via [ActivityPub](https://activitypub.rocks/) para que o blog também seja acessível a partir de outras plataformas federadas.

---

## Linha Editorial e Manifesto

Este blogue surge para contribuir com um debate público plural, ético, laico e estritamente fundamentado em evidências e na integridade científica.

Nossos pilares fundamentais são:

- **Integridade Científica e Esclarecimento**: abordar temas complexos de forma acessível e baseada no melhor consenso científico disponível.
- **Laicidade e Pluralidade**: oferecer um espaço livre de dogmas, aberto ao debate racional e respeitoso.
- **Combate Ativo à Desinformação**: investigar fontes e desmistificar falsos consensos ou boatos infundados.
- **Cultura Livre e Software Livre (FLOSS)**: acreditar no poder da descentralização, do código aberto e do acesso universal ao conhecimento.

## Tecnologias Utilizadas

O blog foi construído utilizando as seguintes ferramentas modernas de desenvolvimento web:

- **Astro**: framework web focado em performance absoluta e entrega de páginas HTML estáticas ultrarrápidas.
- **MDX**: permite escrever posts utilizando Markdown integrado com componentes dinâmicos e estilizados.
- **Tailwind CSS**: design moderno, responsivo e minimalista.
- **Giscus**: sistema de comentários moderno baseado em Discussões do próprio GitHub do repositório.
- **GitHub Actions**: pipeline de integração contínua para compilar e publicar o site automaticamente no GitHub Pages.
- **Template Base**: desenvolvido a partir do elegante tema open-source [Serene Ink](https://github.com/jiseeeh/serene-ink) para Astro.
